Project Overview

NBFC loan portfolios involve multiple products, tenors, security structures, and borrower identifiers, making monitoring and analysis complex at scale. Manual or fragmented reporting often limits timely risk identification and management decision-making.

This dashboard was designed to consolidate loan book data into a single, structured analytics layer, enabling portfolio monitoring, trend analysis, exception detection, and data integrity validation.

Key Analytics Coverage

Loan Disbursement Analysis

  • Month-on-month disbursement trends
  • Product-wise and tenor-wise analysis
  • Disbursement timing patterns

Interest Rate & Revenue

  • Rate of interest distribution
  • Product-level yield analysis
  • Revenue contribution by portfolio segment

NPA & Credit Risk

  • NPA identification and tracking
  • Product-wise and branch-wise NPAs
  • Historical comparison across periods

Security & Collateral Analysis

  • Security type classification
  • Collateral value monitoring
  • Loan-to-value (LTV) analysis

UCIC & Data Integrity Checks

  • UCIC-PAN uniqueness validation
  • Duplicate customer detection
  • Master data consistency checks

Geographic & Branch Insights

  • Branch-wise loan distribution
  • Regional exposure analysis
  • Portfolio concentration monitoring
